## Overview

DrinkMe is a minimalist desktop app that **automatically compresses** videos into smaller filesâ€”no manual settings or technical know-how required. Just pick a file, click **Shrink**, and save space.

---

## Features

- Automatic, two-pass compression for best quality/size
- File estimation size before shrinking
- Warning if savings are under 10 MB

---

## Usage

```bash
# 1. Launch DrinkMe
# 2. Click "Select File" and choose your video
# 4. Click "Shrink"
# 5. Find your compressed video alongside the original
```
